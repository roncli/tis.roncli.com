Why are there SongIDs missing?
There are two possible reasons for this. First, the song could have been removed from Trax in Space, either by the artist's intentions or by the file being corrupt. In either case, the song was simply deleted. Second, there were big gaps in the 14000 series caused by problems with the automatic uploader, which caused many SongIDs to go unused.

What do these files that have lost.txt mean?
There are 2 known songs that I could not upload to the server, due to CRC errors from the original CD. These are SongIDs 15008 and 15009. The files are still in place, but instead contain a .txt file indicating that the file has been lost, possibly forever.

Can I contribute to this archive?
